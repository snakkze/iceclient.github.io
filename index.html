About Ice

Ice is a open-source Java Minecraft Client created with Forge-Gradle and Mixins.

Features

We mainly try to concentrate on new, individual features, while also trying keeping it as vanilla-like as possible, so you can experience an enhanced Minecraft, and not a new game.
